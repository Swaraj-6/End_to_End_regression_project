Rules for Uploading Training Data Set

There are 12 Rules for Uploading Training Data Set

  • Uploaded file must be a csv file

  • File must have name as ("upload.csv")

  • File must have 10 independent variable and 1 dependent variable

  • First column must be an ID of unique number for each row

  • Second column must be carat of diamond in float formate

  • Third column must be of cut with categorized as Fari, Good, Very Good, Premium, Ideal with Fair being lowest and Ideal being of higest value

  • Forth column must be of color with categorizes as D, E, F, G, H, I, J with D being lowest and J being of highest value.

  • Fifth column must be of clarity of diamond with categorieze as I1, SI2, SI1, VS2, VS1, VVS2, WS1, IF with I1 being lowest and IF being of highest value.

  • Sixth column must be of bepth of diamond in mm.

  • Seventh column must be the diamond's table in facet.

  • Eighth, Ninth & Tenth column must be diamond's X,Y,Z dimension.

  • Eleventh column must be the price of the diamond in Dollar($).
